# 🎓 Student Grade Management System

## 📌 Description

This is a simple command-line based Student Grade Management System built using Python.  
It allows users to enter marks for multiple subjects, validates the input, calculates the average, and assigns a grade based on performance.

---

## 🚀 Features

* Accepts marks for 5 subjects
* Validates input (marks must be between 0 and 100)
* Calculates average using a function
* Assigns grades (A, B, C, D, F)
* Handles invalid inputs using exception handling
* Modular and clean code using functions

---

## 🛠️ Technologies Used

1. Install Python
2. Download or clone this repository
